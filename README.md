## 项目名称：基于“58 同城”平台的上海二手房信息的决策分析
### 项目概述：
2021 年是二手房市场迎来大变革的一年，二手房价格参考制度成为全新的调控工具，对市场交易和预期都产生巨大影响。作为国内领先的房产信息及交易服务平台，我们通过对58 同城的二手房信息搜集整理，通过对数据进行数据清理、可视化以及通过数据挖掘对上海二手房进行预测,从住房者关注的多方面进行分析,为有意向的相关房客提供借鉴和参考。
该项目通过使用 Scrapy 爬虫框架、数据采集和数据清洗使用 Pandas 工具、MySQL 数据库、中文分词组件 Jieba、数据可视化框架以及一些代码格式化工具等，对“58 同城”的上海二手房的信息数据进行采集、清洗、分析最终生成 CSV 表的形式并导入到MySQL 数据库中，同时，为了使最终得到的数据能够更加清晰直观，通过构建 Django 开发框架，使得导出到 MySQL 中的数据在页面上能够以图表的形式进行直观展示查看并进行分析。最后的数据挖掘部分通过构建出最佳的决策树回归分析模型以实现用户根据自身需求预测出大致房源总价，选择合适房源的目的。
